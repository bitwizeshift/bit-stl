6 /*
7  * Symbols Defined:
8  *
9  * BIT_STRINGIZE(a) : Converts symbol 'a' to string at compile time.
10  * BIT_JOIN(lhs,rhs) : concatenates lhs to rhs at compile time, producing symbol lhsrhs.
11  * BIT_TO_BOOL(x) : Converts integer x to a boolean value
12  * BIT_BOOL_NOT(x) : Inverts the boolean value (1 to 0 or 0 to 1)
13  * BIT_BOOL_OR(x,y) : Performs bitwise or, returning the result of x||y
14  * BIT_BOOL_AND(x,y) : Performs bitwise and, returning the result of x&&y
15  * BIT_BOOL_XOR(x,y) : Performs bitwise xor, returning the result of x^y
16  * BIT_BOOL_NAND(x,y) : Performs bitwise nand, returning the result of !(x&&y)
17  * BIT_BOOL_NOR(x,y) : Performs bitwise nor, returning the result of !(x||y)
18  * BIT_BOOL_XNOR(x,y) : Performs bitwise xnor, returning the result of !(x^y)
19  * BIT_IF(cond,x) : If condition is true, perform x
20  * BIT_IF_ELSE(cond,x,y) : If condition is true, perform x, otherwise y
21  * BIT_COMMA_IF(cond) : If condition, create comma
22  * BIT_LIST(c,p) : Creates an increasing list of c elements, prefixed by 'p'
23  * BIT_REVERSE_LIST(c,p) : Creates a decreasing list of c elements, prefixed by 'p'
24  * BIT_SELECT(n,...) : Selects the n'th argument of the supplied variadic list
25  * BIT_VA_ARGS(...) : Portable way of passing variadic arguments without a trailing comma on empty input
26  * BIT_EMPTY : Empty symbol for passing to macro parameters to avoid warnings
27  */
