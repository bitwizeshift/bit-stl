template<typename CharT>
struct bit::stl::const_type_wrapper< CharT >

Constructor tag used to differentiate const type pointers from fixed known type arrays at compile time. (e.g. "const T* p" from "const T (&p)[N]")

This is used primarily for methods that need to differentiate known string sizes from pointers at compile time – but can be extended to any generic form of array/pointer
