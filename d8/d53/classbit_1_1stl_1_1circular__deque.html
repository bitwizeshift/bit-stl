Detailed Description

template<typename T, typename Allocator = std::allocator<T>>
class bit::stl::circular_deque< T, Allocator >

A circular buffer with a deque API.

Memory is allocated up-front from the specified allocator

Template Parameters
Tthe underlying type
Allocatorthe allocator type
