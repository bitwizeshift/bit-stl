Detailed Description

template<typename T>
class bit::stl::expected< T >

An expected type is a combination of an optional and an exception type.

Rather than supporting a null-case like optional, expected either contains an instance of a value, or an exception.

If it is an exception, and the value is attempted to be accessed, it will instead throw the stored exception.

The type of the exception can also be queried, allowing for a query- based API rather than forcing stack-unwinding
