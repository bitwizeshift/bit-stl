Detailed Description

template<typename Iterator, typename U = void>
class bit::stl::tagged_iterator< Iterator, U >

A thin wrapper around a different iterator type to add uniqueness.

This makes any regular Iterator type such as a pointer to be unique, and incomparable to other pointers.

Template Parameters
Iteratorthe pointer type to wrap
Ua type to make this iterator unique, and not comparable with other tagged_iterators
