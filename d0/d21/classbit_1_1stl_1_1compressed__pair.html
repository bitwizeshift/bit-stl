Detailed Description

template<typename T1, typename T2>
class bit::stl::compressed_pair< T1, T2 >

compressed_pair is a struct template that provides a way to store two heterogeneous objects as a single unit that leverages empty-base-optimization

A compressed pair is a special case of a compressed_tuple with only two elements.
