Detailed Description

template<typename Enum>
struct bit::stl::enum_traits< Enum >

Traits for an enum.

Implementations should provide specialiations of this type to enable to_string and from_string functionality with enum_cast

This struct also must specialize a begin() and end() function that returns type const Enum*, if an enum is to be used for iteration
