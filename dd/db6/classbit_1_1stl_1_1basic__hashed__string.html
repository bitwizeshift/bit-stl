Detailed Description

template<typename CharT, typename Traits = std::char_traits<CharT>, typename Allocator = std::allocator<CharT>>
class bit::stl::basic_hashed_string< CharT, Traits, Allocator >

The basic_hashed_string is a wrapper around an owning basic_string with the hash.

This provides runtime hashed values for std::basic_strings

Since the hash is stored with the string, it allows easy checking of the hash multiple times without any concerns for performance.
