Detailed Description

An exception indicating a bad access was attempted for an expected type.

bit::stl::bad_expected_access is the type of exception thrown in the following situations:

  • expected::value is called on an expected with a null std::exception_ptr
  • expected::value is called on an expected that is valueless_by_exception
