Detailed Description

template<typename Ptr, typename U = void>
class bit::stl::pointer_iterator< Ptr, U >

A thin wrapper around a random-access pointer type.

This makes a regular pointer a unique iterator type, incomparable and not-convertible to normal pointers

Template Parameters
Ptrthe pointer type to wrap
Ua type to make this iterator unique, and not-comparable with other pointer_iterators
