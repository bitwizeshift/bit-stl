Detailed Description

template<typename CharT, typename Traits = std::char_traits<CharT>>
class bit::stl::basic_zstring_view< CharT, Traits >

A wrapper around non-owned zero-terminated strings.

This is an extension to the C++17 string_view

This class is used to interop with C libraries that expect normal strings
