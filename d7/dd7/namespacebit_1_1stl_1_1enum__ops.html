Detailed Description

This namespace introduces bitwise operations to any enums that specialize is_enum_bitmask.

This namespace behaves a lot like std::rel_ops. To make use of this, use this namespace either from within another namespace, or from the top level. These overloads will only selectively enable if bit::stl::is_enum_bitmask has been specialized to be true for that enum.
