Detailed Description

template<typename CharT, typename Traits = std::char_traits<CharT>>
class bit::stl::basic_hashed_string_view< CharT, Traits >

The basic_hashed_string_view is a compile-time hashed wrapper around a non-owning string (basic_string_view).

This provides compile-time hashed values for string literals, or constant-time hashed values for non-literal strings.

Since the hash is stored with the string, it allows easy checking of the hash multiple times without any concerns for performance.
